# 4. 기본 문법

**4.1 세미콜론**

- 라인이 끝날 때 세미콜론을 쓰지 않아도 된다.

```python
print('Hello, world!')
```

- 한 줄에 여러 구문을 사용할 때 세미콜론을 사용한다.

```python
print('Hello, world!'); print('python');
```



**4.2 주석**

```python
# 한 줄 주석

# 여러줄
# 주석
```



**4.3 들여쓰기**

- 들여쓰기가 문법이다.

```python
if a == 10:
    print('10입니다.')
```



**4.4 코드 블럭**

- 코드 블럭은 특정한 동작을 위하여 코드가 모여 있는 상태를 뜻한다.
- 같은 블럭은 들여쓰기 칸 수가 같아야 한다.
- 공백과 탭 문자를 섞어 들여쓰기를 하면 안 된다.

```python
if a == 10:
      print('10')
      print('입니다.')
```
