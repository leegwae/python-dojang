## 1. print() 함수를 사용한 다양한 출력

```python
print("%서식" % 값1)
print("%서식1 %서식2" % (값1, 값2))
```



```python
print("%d" % 30)		# 30
print("%d / %d = %d" % (3, 2, 3/2))		# 3 / 2 = 1
```



**print() 함수에서 사용할 수 있는 서식**

| 서식             | 값의 예        | 의미                        |
| ---------------- | -------------- | --------------------------- |
| `%d`, `%x`, `%o` | 10, 100, 1234  | 정수(10진수, 16진수, 8진수) |
| `%f`             | 0.5, 1.0, 3.14 | 실수(소수점이 붙은 수)      |
| `%c`             | "b", "한"      | 한 글자                     |
| `%s`             | "안녕", "a"    | 두 글자 이상의 문자열       |



**정수형 데이터의 서식 지정**

```python
print("%d" % 123)		# 123
print("%5d" % 123)		#   123
print("%05d" % 123)		# 00123
```

- `%d`: 값의 자릿수만큼 그대로 출력
  - 123
- `%5d`: 5만큼의 자릿수를 확보하고 오른쪽으로 정렬하여 출력
  - {빈칸}{빈칸}123
- `%05d`: 5만큼의 자릿수를 확보하고 오른쪽으로 정렬, 빈칸은 0으로 채워 출력
  - 00123



**실수형 데이터의 서식 지정**

```python
print("%f" % 123.45)		# 123.450000
print("%7.1f" % 123.45)		#   123.5
print("%7.3f" % 123.45)		# 123.450
```

- `%f`: 소수점 아래 6자리 출력
- `%7.1f`: 소수점을 포함하여 7만큼의 자릿수를 확보하고, 오른쪽으로 정렬하여 소수점 아래 2자리에서 반올림하여 소수점 아래 1자리까지 출력. 
  - {빈칸}{빈칸}123.5
- `%7.3f`
  - 123.450



**문자열 데이터의 서식 지정**

```python
print("%s" % "Hello")	# Hello
print("%10s" % "Hello")	#      Hello
```

- `%s`: 문자열을 그대로 출력
- `%10s`: 10만큼의 자릿수를 확보하고 오른쪽으로 정렬



**format() 함수를 이용하여 서식 지정하기**

```python
print("{순서1:서식1} {순서2:서식2} {순서3: 서식3}".format(값1, 값2, 값3))
```



```python
print("{0:d} {1:5d} {2:05d}".format(123,123,123))
# 123   123 00123
```

```python
print("{2:d} {1:d} {0:d}".format(100,200,300))
# 300 200 100
```

**제어 문자**

| 이스케이프 문자 | 의미      |
| --------------- | --------- |
| `\n`            | new line  |
| `\t`            | tab       |
| `\n`            | backspace |
| `\\`            | `\` 출력  |
| `\'`            | `'` 출력  |
| `\"`            | `"` 출력  |

- `\b`는 OS에 따라 다르게 작동한다. Windows에서는 제대로 작동하지 않는다.

```python
print("Hello, world\b");
# 출력 : Hello, worl
```





**제어 문자 무시하기**

```python
print(r"\n ㅎㅎ")
# \n ㅎㅎ
```

- `print()` 함수의 파라미터 앞에 `r`을 붙인다.

## 3. 변수의 선언과 사용

**변수의 선언**

```python
a = 1
b = "안녕"
a = "삵"
```

- 변수는 값이 할당될 때마다 새로이 데이터 형식이 변경될 수 있다. 여기서 `a`는 정수에서 문자열로 데이터 형식이 변경된다.



**변수 여러 개 선언**

```python
a, b = 1, "안녕"
```



**type() 함수로 변수의 타입 확인하기**

```shell
>>> type("안녕")
<class 'str'>
```



**변수 이름 짓기**

- 대/소문자를 구분한다.
- 숫자로 시작할 수 없다.
- 예약어를 변수명으로 쓰면 안된다.
