# Keyword

| 키워드                  | 의미                                                         | 참고   |
| ----------------------- | ------------------------------------------------------------ | ------ |
| `pass`                  | 아무것도 실행하지 않고 넘어간다.                             | 13.1.3 |
| `값 in 시퀀스 객체`     | 값이 시퀀스 객체 안에 있는지 확인하고 있으면 `True`를 반환한다. | 11.1.1 |
| `값 not in 시퀀스 객체` | 값이 시퀀스 객체 안에 있는지 확인하고 없으면 `True`를 반환한다. |        |
