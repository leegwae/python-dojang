# PEP 8 - Style Guide for Python Code

https://www.python.org/dev/peps/pep-0008/



## 네이밍 컨벤션

- 모듈: 짧은 소문자. 밑줄로 나누기
- 클래스명: 카멜케이스. 내부적이면 밑줄로 시작.
  - 예외는 뒤에 `Error`를 붙이기
- 함수명: 소문자. 밑줄로 나누기
- 인스턴스 메서드의 첫번째 인자: `self`
- 클래스 메서드의 첫번째 인자: `cls`
- 메서드명: 함수명과 같다. non-public 메서드/변수이면 밑줄로 시작
- 서브 클래스: 밑줄 두 개로 시작
- 상수: 모듈 단위에서 정의, 대문자, 밑줄로 나누기
- 변수명: 스네이크 케이스
